Headings: HTML provides 6 headings which are by default bold and Each one varies in their size and importance
<h1>context</h1>
<h2>context</h2>
<h3>context</h3>
<h4>context</h4>
<h5>context</h5>
<h6>context</h6>

we can have extensions such as Prettier for code formatting and Material Icon Theme for Each file or folder Icons

All headings sizes are different (in decreasing order). h1 is the highest one with more importance and h6 is the smallest one with low importance. For better SEO, it is recommanded to use one h1 per Page

Paragraph: Paragraphs are used to define normal context. we use 'p' tag for This
<p>context</p>

lorem is the emmet used for generating dummy paragraphs. lorem10 will generate 10 words paragraph


Anchor Tag: anchor is used to link multiple pages or any other links. we use 'a' tag for This
